FTP 状态    FTP 帐号    导入/导出
 FTP帐号列表:  
 帐号信息:
       帐 号 名:     (帐号不区分大小写)
       帐号密码:     (如果不设置密码则表示无需密码)
       密码保护:不加密 MD4 MD5   (如果设了密码保护则FTP客户端也要相应设置)
       帐号描述:
       有效期限:(yyyy-mm-dd) 永不过期
       登陆限制:     (最多允许几个人使用此帐号同时登陆,0则不限)
       上载限制:KB/s (最大上载速度限制,0则不限制上载速度)
       上载限制:KB   (限制上载文件大小,0则不限制上载大小)
       下载限制:KB/s (最大下载速度限制,0则不限制下载速度)
       限制最大磁盘使用空间:
           允许最大可用空间:MB (0则不限制可用空间)
           当前用户已用空间:MB (0则没有使用任何空间)
       显示隐藏的目录和文件          禁用此帐号
 帐号访问权限:  
       主 目 录:     (绝对物理路径,为空则可访问整个电脑)
       访问权限:文件读 写 删除 执行 目录List 创建 删除 禁止继承
       虚 目 录:       
       虚目录名:      物理路径:
       访问权限:文件读 写 删除 执行 目录List 创建 删除 禁止继承
 IP 过滤:  
       访问许可: 允许访问 禁止访问
       IP 地 址:
       注:可输入多个IP地址,IP地址之间以逗号分割.IP地址支持*通配符号.
          例如: 192.168.0.12,192.169.1.*,192.166.*.5
          *仅用来匹配IP地址4段号码中的一个段.下列写法不支持192.168.1.1*
