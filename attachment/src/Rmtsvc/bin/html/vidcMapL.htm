映射服务列表: 类型:TCP UDP
 映射服务状态:  
       ●服务停止运行
       服务开始运行时间 YYYY年MM月DD日 hh:mm:ss
       当前共 0 个连接本映射服务

 映射服务信息:
       应用服务地址:         应用服务端口:
       应用服务描述:
       应用服务类型:自动检测 普通TCP服务 WWW服务 FTP服务

       本地映射端口:         绑定本地 IP:
       注:映射端口如果设为0则自动分配,如果本机有多个IP可将映射服务绑定到指定的IP
       SSL服务转换: 不进行转换 转换为SSL加密服务 转换为非SSL加密服务
       注:此选项可将SSL加密应用服务转换为普通服务,将普通服务转换为SSL加密服务
       SSL证书验证:不启用客户端证书验证      启用客户端证书验证

       最大并发连接:         最大带宽限制:KB/s (0则不限制)
       映射服务名称:         程序运行自动启动本映射服务

 映射服务过滤:  
       访问许可: 允许访问 禁止访问
       IP 地 址:
       注:可输入多个IP地址,IP地址之间以逗号分割.IP地址支持*通配符号.
          例如: 192.168.0.12,192.169.1.*,192.166.*.5
          *仅用来匹配IP地址4段号码中的一个段.下列写法不支持192.168.1.1*
